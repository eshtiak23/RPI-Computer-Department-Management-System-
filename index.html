About Department

A practical place for future IT professionals.

CST focuses on programming, web development, database management, networking, computer architecture, software engineering, and industry-ready technical skills.

💻

Programming

Learn problem solving, C/C++, Java, Python basics, and structured programming logic.

🌐

Web & Database

Build responsive websites, manage databases, and understand real-world application systems.

🛡️

Networking

Study LAN, WAN, routing, switching, server basics, and network troubleshooting.
